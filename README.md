# Консольный калькулятор на C#

## Описание проекта
Простой консольный калькулятор, написанный на языке C#, который поддерживает базовые арифметические операции, операции остатка от деления, инкремента и декремента.

## Функциональность
- Сложение, вычитание, умножение, деление
- Операция остатка от деления (%)
- Унарные операции инкремента (++) и декремента (--)
- Поддержка десятичных чисел (с точкой или запятой)
- Обработка ошибки деления на ноль

## Особенности
- **Поддержка UTF-8** для корректного отображения символов
- **Гибкий ввод чисел** - принимает как с точкой, так и с запятой
- **Два режима работы**:
  - Бинарные операции (требуют два числа)
  - Унарные операции (инкремент/декремент, требуют одно число)
- **Защита от деления на ноль**

## Требования
- .NET 6.0 или выше
- Любая ОС с поддержкой .NET (Windows, macOS, Linux)
